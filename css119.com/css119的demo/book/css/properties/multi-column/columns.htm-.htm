columns

  • 版本：CSS3
  • 继承性：无

语法：

columns：[ column-width ] || [ column-count ]

默认值：看每个独立属性

取值：

[ column-width ]：
设置或检索对象每列的宽度
[ column-count ]：
设置或检索对象的列数

说明：

设置或检索对象的列数和每列的宽度。复合属性
  • 对应的脚本特性为columns。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0-9.0 4.0-8.0 5.1 13.0-16.0 11.50
版本 10.0 9.0

写法：

内核类型 写法
Webkit(Chrome/Safari) -webkit-columns
Gecko(Firefox) -moz-columns
Presto(Opera) columns
Trident(IE) columns
W3C columns
