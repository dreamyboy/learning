page-break-before

  • 版本：CSS2
  • 继承性：无

语法：

page-break-before：auto | always | avoid | left | right

默认值：auto

取值：

auto：
如果需要，在对象之前插入页分割符
always：
始终在对象之前插入页分割符
avoid：
避免在对象前面插入页分割符
left：
在对象前面插入页分割符直到它到达一个空白的左页边
right：
在对象前面插入页分割符直到它到达一个空白的右页边

说明：

检索或设置对象之前出现的页分割符。
  • 对应的脚本特性为pageBreakBefore。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6.0 4.0 5.1 13.0 11.50
