text-transform

  • 版本：CSS1/CSS3
  • 继承性：有

语法：

text-transform：none | [ [ capitalize | uppercase | lowercase ] || full-width || full-size-kana ]

默认值：none

取值：

none：
无转换
capitalize：
将每个单词的第一个字母转换成大写
uppercase：
转换成大写
lowercase：
转换成小写
full-width：
??（CSS3）
full-size-kana：
??（CSS3）

说明：

检索或设置对象中的文本的大小写。
  • 对应的脚本特性为textTransform。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0 4.0 5.1 13.0 11.50
