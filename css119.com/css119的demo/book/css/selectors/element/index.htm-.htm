* 通配选择符(Universal Selector) CSS2 所有元素对象。
E 类型选择符(Type Selector) CSS1 以文档语言对象类型作为选择符。
E#myid id选择符(ID Selector) CSS1 以唯一标识符id属性等于myid的E对象作为选择符。
E.myclass class选择符(Class Selector) CSS1 以class属性包含myclass的E对象作为选择符。
