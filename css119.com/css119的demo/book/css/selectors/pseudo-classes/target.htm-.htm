目标伪类选择符 E:target

  • 版本：CSS3

语法：

E:target{ sRules }

说明：

匹配相关URL指向的E元素。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6.0-8.0 4.0 5.1 13.0 11.50
较新版本 9.0
