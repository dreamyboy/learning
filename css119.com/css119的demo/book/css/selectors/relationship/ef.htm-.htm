包含选择符 (E F)

  • 版本：CSS1

语法：

E F{ sRules }

说明：

选择所有被E元素包含的F元素。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0 4.0 5.1 13.0 11.50
