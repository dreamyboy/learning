pt

  • 版本：CSS1

说明：

点（Points）。绝对长度单位。
  • 1in = 2.54cm = 25.4 mm = 72pt = 6pc = 96px

示例代码：

div{font-size:10pt;}
p{height:100pt;}

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0 4.0 5.1 13.0 11.50
