counter()

  • 版本：CSS2

说明：

插入计数器。
  • 在CSS2.1中counter()只能被使用在content属性上。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
支持版本\类型 IE Firefox Safari Chrome Opera
版本 6.0-7.0 4.0 5.1 13.0 11.50
版本 8.0
