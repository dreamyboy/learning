<length>

  • 版本：CSS1

说明：

<number>接长度单位。
  • 特殊值0可以省略单位。例如：margin:0px可以写成margin:0
  • 一些属性可能允许有负长度值，或者有一定的范围限制。如果不支持负长度值，那应该变换到能够被支持的最近的一个长度值。
  • 长度单位包括包括：相对单位和绝对单位。
  • 相对长度单位包括有：em, ex, ch, rem, vw, vh, vm, %
  • 绝对长度单位包括有：cm, mm, in, pt, pc, px

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 墨绿 = 部分支持
  • 橙色 = 实验性质
